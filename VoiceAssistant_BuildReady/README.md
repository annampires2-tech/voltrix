## Installation

### Requirements
- Android 7.0 or higher
- Microphone permission
- Phone/SMS permissions

### Build from source
```bash
cd ~/VoiceAssistant
chmod +x build.sh
./build.sh
```

### Install on phone
```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

Or transfer the APK to your phone and install manually.

## Usage
1. Open the app
2. Grant all permissions
3. Tap "Start Listening"
4. Speak your command
5. Assistant will respond and execute

## Customization

### AI Provider Options (FREE alternatives!)

**Default: Groq (FREE & FAST)**
- No API key needed initially
- Get free key at https://console.groq.com
- Uses Llama 3.1 70B model
- Very fast responses

**Voice commands:**
- "Use Groq" (default, free)
- "Use OpenAI" (requires paid API key)
- "Use Ollama" (local, completely offline)
- "Set API key [your-key]"

**Setup Groq (Recommended - FREE):**
1. Go to https://console.groq.com
2. Sign up (free)
3. Get API key
4. Say: "Set API key [your-groq-key]"

**Setup Ollama (Offline AI):**
1. Install Ollama on your phone/PC
2. Run: `ollama run llama3`
3. Say: "Use Ollama"
4. Completely offline!

**Setup OpenAI (Paid):**
1. Get key from https://platform.openai.com
2. Say: "Set API key [your-openai-key]"
3. Say: "Use OpenAI"
